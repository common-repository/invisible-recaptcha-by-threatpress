A simple plugin to integrate the new Invisible reCAPTCHA into your WordPress site.

== Description ==

[Google reCAPTCHA](https://www.google.com/recaptcha/intro/invisible.html) is designed to protect your WordPress site from spam bots. This plugin integrates [the new Invisible reCAPTCHA](https://www.google.com/recaptcha/intro/invisible.html) into your WordPress site.

**What is protected with Invisible reCaptcha?**

**WordPress**
* Login form
* Registration form
* Comments form
* Forgot password form

**WooCommerce**
* Login form
* Registration form
* Lost password form
* Reset password form
* Product review form

== Installation ==
1. Upload `invisible-recaptcha-by-threatpress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the API keys

== Frequently Asked Questions ==

= How to get Google Invisible reCAPTCHA API keys?
1. Log in to [Google reCAPTCHA web site](https://www.google.com/recaptcha/intro/index.html)
2. Register a new site
3. Choose the type of reCAPTCHA to Invisible reCAPTCHA
4. Copy the site and secret key
5. Add the new keys to the plugin
